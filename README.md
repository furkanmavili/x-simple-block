```bash

```

2. Install dependencies

```bash
pnpm install
```

3. Build the extension

```bash
pnpm build
```

4. Load in Chrome:
   - Open Chrome and navigate to `chrome://extensions/`
   - Enable "Developer mode" in the top right corner
   - Click "Load unpacked"
   - Select the `dist` folder from the cloned repository
